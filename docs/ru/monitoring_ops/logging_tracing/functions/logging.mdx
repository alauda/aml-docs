# Ведение журналов

## Обзор возможностей

Функция ведения журналов в Alauda AI позволяет в реальном времени отслеживать и анализировать журналы, создаваемые подами Инференс-сервиса. Эта функциональность помогает пользователям устранять неполадки, отслеживать поведение сервиса и обеспечивать операционную эффективность, предоставляя немедленный доступ к критически важным данным журнала. Она особенно полезна для отладки ошибок вывода модели.

## Основные функции

- **Потоковая передача журналов в реальном времени**: Автоматически отображает журналы по мере их генерации выбранным подом.
- **Выбор пода-реплики**: Переключение между подами для просмотра журналов от конкретных реплик.
- **Поиск по ключевым словам (Найти)**: Найдите конкретные записи журнала, используя точное или частичное соответствие ключевым словам.
- **Экспорт журнала (Экспорт)**: Скачайте журналы в формате `.txt` для офлайн-анализа или архивации.

## Преимущества функции

- **Мгновенная видимость**: Не требуется ручное обновление – журналы обновляются динамически.
- **Поддержка нескольких подов**: Легко переключайтесь между подами для сравнения журналов по репликам.
- **Операционная эффективность**: Встроенные инструменты поиска и экспорта ускоряют анализ первопричин.

## Доступ к журналам

Следуйте этим шагам, чтобы просмотреть журналы для Инференс-сервиса:

<Steps>
  ### Шаг 1: Перейдите к Инференс-сервису

  1. Перейдите в **Инференс-сервисы** в левой панели навигации.
  2. Нажмите на имя целевого Инференс-сервиса, чтобы открыть его страницу с деталями.

  ### Шаг 2: Откройте интерфейс ведения журналов

  1. Выберите вкладку **Ведение журналов** в панели вкладок под именем Инференс-сервиса.

  ### Шаг 3: Выберите под

  1. Используйте выпадающий список **Реплика**, чтобы выбрать под.
  2. Просмотрщик журналов автоматически отобразит и начнет потоковую передачу журналов из выбранного пода.
</Steps>

## Использование функции Найти

Искать журналы эффективно, используя соответствие ключевым словам:

<Steps>
  ### Шаг 1: Активируйте поиск

  1. Нажмите кнопку **Найти** в правом верхнем углу просмотрщика журналов.

  ### Шаг 2: Введите поисковый запрос

  1. Введите ключевое слово или фразу в поле поиска.

  ### Шаг 3: Перейдите к результатам

  1. Совпадающие записи выделены желтым цветом.
  2. Используйте кнопки **↑** и **↓**, чтобы перейти между совпадениями.
</Steps>

## Экспорт журналов

Экспортируйте журналы для офлайн-хранения и анализа:

<Steps>
  ### Шаг 1: Инициируйте экспорт

  1. Нажмите кнопку **Экспорт** в правом верхнем углу просмотрщика журналов.

  ### Шаг 2: Скачайте файл

  1. Браузер автоматически скачает файл журнала на вашу локальную систему.
</Steps>
