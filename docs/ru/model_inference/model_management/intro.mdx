# Введение

Репозиторий моделей представляет собой версионированный хранилище и центр управления, разработанный специально для моделей LLM на платформе Alauda AI. Основанный на технологии Git LFS, он предоставляет эффективное и надежное решение для хранения моделей, охватывающее критически важные этапы, такие как развертывание услуг вывода, дообучение моделей и оценка моделей. Через репозиторий моделей пользователи могут легко управлять версиями моделей, делиться активами моделей и достигать полного управления жизненным циклом моделей.

## Основные преимущества

- **Точное управление версиями:**
  - Использует возможности ветвления и тегирования Git для эффективного управления версиями моделей, позволяя пользователям отслеживать и откатывать изменения моделей.
- **Эффективное хранение больших файлов:**
  - Использует технологию Git LFS для оптимизации хранения и передачи больших файлов моделей, что улучшает скорость загрузки и скачивания моделей.
- **Гибкое совместное использование моделей:**
  - Поддерживает как частные, так и общие режимы, удовлетворяя разнообразные потребности в обмене моделями и облегчая обмен моделями внутри и вне предприятия.
  - Благодаря настройкам пространства имен и совместного использования легко создать рынок моделей и упростить распределение моделей.
- **Бесшовная интеграция рабочего процесса:**
  - Глубоко интегрируется с развертыванием услуг вывода, дообучением моделей и функциями оценки, упрощая процессы развертывания и применения моделей.

## Сценарии применения

- **Управление версиями моделей:**
  - Контроль версий для моделей LLM, обеспечивающий отслеживаемость изменений моделей.
- **Совместное использование активов моделей:**
  - Совместное использование моделей внутри команд или предприятий для повышения повторного использования и сотрудничества по моделям.
  - Создание рынка моделей, упрощение внешнего распределения моделей.
- **Развертывание и применение моделей:**
  - Служит источником моделей для развертывания услуг вывода, упрощая процесс развертывания моделей.
  - Функционирует как центр хранения для дообучения и оценки моделей, что облегчает итерацию моделей.
