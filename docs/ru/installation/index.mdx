# Установка

## Аппаратные требования

- По крайней мере два узла с 16 ядрами и 32 ГБ общей памяти.
- Дополнительные ресурсы для выполнения в реальном времени определяются фактическим масштабом бизнеса: 10 экземпляров вывода LLM размером 7B одновременно требуют как минимум 10 графических процессоров (GPU), а также соответствующие центральный процессор (CPU), память, дисковое пространство и объектное хранилище.
- 200 ГБ свободного дискового пространства для каждого узлового работника.

## Предварительные требования

Вы должны соответствовать следующим требованиям, прежде чем вы сможете установить Alauda AI на вашем кластере ACP:

- **Доступ администратора к вашему кластеру ACP**
- **MySQL 5.7 (или лучше)**
- **Service Mesh с Istio 1.20 (или лучше)**
- **Gitlab (саморазмещенный) v15 или лучше**
