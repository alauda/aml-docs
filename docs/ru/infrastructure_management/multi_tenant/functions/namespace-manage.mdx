# Управление пространствами имен

Тенанты, поддерживаемые AML, представляют собой пространства имен K8s, которые необходимо управлять в **Управлении пространствами имен** перед их использованием как тенанты AML.

Процедура создания пространства имен и добавления пользователей в это пространство имен выглядит следующим образом:

1. Войдите в платформу ACP как администратор

2. Создайте пространство имен
   1. Перейдите в **Управление проектами** ACP
   2. Создайте новый проект или выберите существующий, затем нажмите, чтобы перейти на страницу сведений о текущем проекте.
   3. Нажмите **Пространство имен** > **Пространство имен**
   4. Нажмите **Создать пространство имен**
   5. На странице **Создание пространства имен**, в разделе **Политики безопасности подов**, выберите **привилегированный** в качестве стандарта безопасности для всех трех режимов безопасности `enforce`, `audit` и `warn`.

      :::warning

      Стандарт безопасности должен быть либо пустым, либо **привилегированным**, в противном случае рабочая нагрузка AI Inference может отказаться от выполнения.

      :::

3. Назначьте права на пространство имен пользователям
   1. Перейдите в **Управление платформой** ACP
   2. Выберите **Управление ролями пользователей** > **Управление пользователями** из навигации
   3. Создайте нового пользователя или выберите пользователя, которого нужно добавить в это пространство имен
   4. Перейдите на вкладку **Настроить роль** и нажмите кнопку **Добавить роль**
   5. Настройте роли, связанные с AML, для пользователя и выберите ранее созданное пространство имен
      - Редактор: Пользователи могут создавать и удалять рабочие нагрузки в пространстве имен и использовать ресурсные квоты, такие как CPU и GPU, для выполнения работы
      - Владелец: Пользователи имеют право управлять этим пространством имен AML
      - Просмотрщик: Пользователи могут только просматривать ресурсы пространства имен и не могут создавать или удалять рабочие нагрузки

4. Вернитесь в AML, нажмите **Управление пространством имен** в **Управлении платформой** > **Управлении пространствами имен** и выберите ранее созданное пространство имен для завершения управления
