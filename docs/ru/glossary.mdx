| название              | Описание                                                                                                                                                                                                                                                                                                   |
|----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Большая языковая модель | LLM (Большая языковая модель) - это ИИ-модель, обученная на огромных объемах текстовых данных, способная понимать и генерировать естественный язык, с масштабом параметров, обычно варьирующимся от миллиардов до сотен миллиардов.                                                                 |
| Сервис вывода        | Сервис вывода - это услуга в области машинного обучения или глубокого обучения, которая предоставляет высокопроизводительные, масштабируемые возможности предсказания или вывода для обученных моделей.                                                                                                     |
| Исполнения вывода    | Исполнение вывода предоставляет высокопроизводительную контейнерную среду для услуг вывода на основе моделей, оптимизируя использование ресурсов, ускоряя процесс вывода и уменьшая задержку.                                                                                                            |
| ИИ-агент             | ИИ-агент - это ИИ-существо, которое воспринимает свою среду, принимает решения и выполняет задачи автономно, обладая автономностью, адаптивностью и ориентированностью на цели.                                                                                                                       |
| Генерация текста     | Генерация текста - это процесс автоматической генерации когерентного и значимого текстового контента с использованием технологий обработки естественного языка (NLP), основанный на заданных входных данных, таких как контекст, подсказки или правила.                                               |
| Классификация текста | Классификация текста - это процесс присвоения текстовым данным заранее определенных категорий или меток, который, как правило, осуществляется с помощью моделей машинного обучения или глубокого обучения и используется в различных приложениях, таких как извлечение информации, анализ настроений и обнаружение спама. |
| Текст-в-изображение  | Текст-в-изображение - это процесс автоматической генерации изображений из текстовых описаний с помощью технологий ИИ, которые объединяют NLP и компьютерное зрение для преобразования текста в визуальный контент.                                                                                                    |
| Виртуальный GPU      | Виртуальный GPU (vGPU) - это технология, которая использует виртуализацию для разделения и распределения физических ресурсов GPU между несколькими виртуальными машинами, что позволяет эффективно использовать графические вычислительные возможности.                                                                |
| Физический GPU       | Физический GPU (pGPU) - это технология, которая непосредственно соединяет всю физическую видеокарту на хост-машине с виртуальной машиной для эксклюзивного доступа к ресурсам GPU.                                                                                                                    |
