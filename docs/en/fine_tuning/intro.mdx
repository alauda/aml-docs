# Introduction

This document introduces a cloud-native visual training platform for fine-tuning large language models (LLMs). Through an intuitive Web UI, users can complete data preparation, fine-tuning configuration, task submission, monitoring, and evaluation without writing code. The platform supports mainstream open-source LLMs (such as LLaMA, ChatGLM, and Qwen), integrates distributed training, resource scheduling, log tracking, and model version management, and is designed to scale reliably for enterprise-level AI research and development.
