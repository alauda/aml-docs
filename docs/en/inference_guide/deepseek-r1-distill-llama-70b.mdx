# DeepSeek-R1-Distill-Llama-70B (dense)

Dense 70B model (Llama 3.3 base, R1-reasoning distilled, BF16, ~141 GB). Validated on Ascend
910B3 at TP=8 (a full 8-card node, one replica) with vLLM-Ascend, with accuracy and
single-stream perf measured.

## Model identity

| Field | Value |
