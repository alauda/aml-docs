# Qwen3-14B (dense)

Dense 14B model (BF16, ~29.5 GB — fits a single Ascend 910B3 card). Validated on Ascend
910B3 at TP=1 and TP=2 with both vLLM-Ascend and MindIE.

## Model identity

| Field | Value |
|---|---|
| Publisher | Qwen (Alibaba) |
| Architecture | `Qwen3ForCausalLM` (dense) — 40 layers / 40 heads / hidden 5120 |
| Parameters | 14B |
| Native dtype | BF16 (non-quantized) |
| Weights size | 29.5 GB (8 safetensors) |
| Hugging Face | https://huggingface.co/Qwen/Qwen3-14B |

## Validated hardware × stack

| Platform | Engine | Version | Status |
|---|---|---|---|
| Ascend 910B3 × 1 (TP=1, whole card) | vLLM-Ascend | `v0.18.0-openeuler` (V1 + FULL_DECODE_ONLY graph) | ✅ 4-scenario open-loop perf |
| Ascend 910B3 × 2 (TP=2) | vLLM-Ascend | same + hostIPC/HCCL | ✅ 4-scenario open-loop perf |
| Ascend 910B3 × 1 (TP=1, whole card) | MindIE | `2.2.RC1` (ATB, auto BF16) | ✅ 4-scenario open-loop perf |
| Ascend 910B3 × 2 (TP=2) | MindIE | same + hostIPC/HCCL | ✅ 4-scenario open-loop perf |
