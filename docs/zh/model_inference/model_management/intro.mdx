# 介绍

模型库是专为 Alauda AI 平台上的 LLM 模型设计的版本化存储和管理中心。它基于 Git LFS 技术，提供高效可靠的模型存储解决方案，涵盖推理服务部署、模型训练微调和模型评估等关键阶段。通过模型库，用户可以轻松管理模型版本，共享模型资产，实现在模型全生命周期的管理。

## 核心优势

- **精准版本控制：**
  - 利用 Git 的分支和标签功能高效管理模型版本，使用户能够追踪和回滚模型更改。
- **高效大文件存储：**
  - 采用 Git LFS 技术优化大模型文件的存储和传输，提升模型上传和下载速度。
- **灵活的模型共享：**
  - 支持私有模式和共享模式，满足不同的模型共享需求，促进企业内外的模型交换。
  - 通过命名空间和共享设置，便于构建模型市场，简化模型分发。
- **无缝工作流程集成：**
  - 深度集成推理服务部署、模型微调和评估功能，简化模型的部署和应用流程。

## 应用场景

- **模型版本管理：**
  - 对 LLM 模型进行版本控制，确保模型更改的可追溯性。
- **模型资产共享：**
  - 在团队或企业内共享模型，促进模型的重用与协作。
  - 构建模型市场，简化对外模型的分发。
- **模型部署与应用：**
  - 作为推理服务部署的模型源，简化模型部署过程。
  - 作为模型微调和评估的存储中心，促进模型迭代。
