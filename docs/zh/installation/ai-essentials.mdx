# Alauda AI Essentials 集群插件

Alauda AI Essentials 集群插件为 Alauda AI 提供了 UI 和 RBAC 资源。

## 下载

:::info
Alauda AI Essentials 集群插件可以从 Alauda 云获取。

如需更多信息，请联系消费者支持。
:::

## 上传

我们需要将 Alauda AI Essentials 集群插件上传到 ACP **global** 集群。

### 步骤

<Steps>
  ### 下载 violet 工具

  首先，如果机器上没有`violet`工具，我们需要下载它。

  在 ACP 控制台的 **平台管理** 中：

  1. 点击 **市场 / 上传包**。
  2. 点击 **下载打包和列表工具**。
  3. 在 **执行环境** 下找到正确的操作系统 / CPU 架构。
  4. 点击 **下载** 以下载 `violet` 工具。
  5. 运行 `chmod +x ${PATH_TO_THE_VIOLET_TOOL}` 来使工具可执行。

  ### 上传包

  然后，运行以下命令将 **Alauda AI Essentials** 集群插件上传到 ACP global 集群：

  ```bash
  violet push \
      ${PKG_NAME} \  # [!code callout]
      --platform-address=${PLATFORM_ADDRESS} \   # [!code callout]
      --platform-username=${PLATFORM_ADMIN_USER} \   # [!code callout]
      --platform-password=${PLATFORM_ADMIN_PASSWORD}   # [!code callout]
      --clusters=global
  ```

  <Callouts>
    1. `${PKG_NAME}` 是 **Alauda AI Essentials** 集群插件的 tarball 名称。
    2. `${PLATFORM_ADDRESS}` 是您的 ACP 平台地址。
    3. `${PLATFORM_ADMIN_USER}` 是 ACP 平台管理员的用户名。
    4. `${PLATFORM_ADMIN_PASSWORD}` 是 ACP 平台管理员的密码。
  </Callouts>
</Steps>

## 安装

### 步骤

在 ACP 控制台的 **平台管理** 中：

1. 从控制台顶部的 **集群** 下拉列表中选择 **global** 集群。
2. 点击 **市场 / 集群插件**。
3. 找到 **Alauda AI Essentials**。
4. 点击 **安装**。

### 验证

确认 **Alauda AI Essentials** 瓦片显示以下状态之一：

- `Installing`: 安装正在进行中；请等待其更改为 `Installed`。
- `Installed`: 安装已完成。
